# Mijn lichaam

##
Ik kan ermee rennen.

##
Ik kan ermee springen.

##
Ik kan ermee dansen.

##
Ik kan ermee zwemmen.

##
Ik kan ermee overspringen.

##
Ik kan ermee trappen.

##
Ik kan ermee ontsnappen.

##
Maar ik kan er nooit mee vliegen.
