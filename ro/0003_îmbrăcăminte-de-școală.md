# Îmbrăcăminte de școală

##
Sarafanul acesta este lung.

##
Jerseul acesta este mare.

##
Geanta aceasta este mare.

##
Cureaua aceasta este....

##
Pălăria aceasta este mică.

##
Șosetele sunt scurte.

##
Dar acești pantofi sunt noi.

##
...se potrivesc bine.
