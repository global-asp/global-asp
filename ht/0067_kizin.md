# Kizin

##
Map kale pòmdetè yo.

##
Map rache yon chou.

##
Map graje karòt yo.

##
Map lave pwa yo.

##
Map koupe yon berejèn.

##
Map rache yon epina.

##
Manman’m ap koupe zonyon yo.

##
Zonyon fè’m kriye lè yo kale yo.
